    **create_thread_logic(description: str, involved_actors: list[str], state: dict) -> dict:**
    - Generate thread_id: `thread_{current_scene}_{keyword}_{index}` where:
      - keyword = first 2-4 Chinese chars from description (extract via regex `[\u4e00-\u9fff]{2,4}`, take first match or "thread" if none)
      - index = count of existing threads with same scene+keyword prefix + 1
    - Create thread dict: id, description, status="active", involved_actors, introduced_scene=current_scene, last_updated_scene=current_scene, progress_notes=[]
    - Validate involved_actors: each must exist in state["actors"], invalid names → return error
    - Append to state["plot_threads"], return {"status": "success", "thread_id": ..., "thread": ...}
    - State is mutated in-place AND returned (following conflict_engine update pattern)

    **update_thread_logic(thread_id: str, status: str | None, progress_note: str | None, state: dict) -> dict:**
    - Find thread by id in state["plot_threads"], return error if not found
    - If status is provided: validate it's one of "active", "dormant", "resolving", "resolved"; update thread["status"]
    - If progress_note is provided: append to thread["progress_notes"] as f"第{current_scene}场：{progress_note}"; FIFO trim to MAX_PROGRESS_NOTES
    - Always update thread["last_updated_scene"] = current_scene
    - Return {"status": "success", "thread_id": ..., "updates_applied": {...}}

    **resolve_thread_logic(thread_id: str, resolution: str, state: dict) -> dict:**
    - Find thread by id, return error if not found or already resolved
    - Set thread["status"] = "resolved"
    - Append f"第{current_scene}场：[解决] {resolution}" to progress_notes (FIFO trim)
    - Check active_conflicts for any with matching thread_id field (D-23): if found, include hint in return value "关联冲突 {conflict_id} 是否也解决？"
    - Return {"status": "success", "thread_id": ..., "resolution": resolution, "linked_conflict_hint": ...}

    **set_actor_arc_logic(actor_name: str, arc_type: str | None, arc_stage: str | None, progress: int | None, related_threads: list[str] | None, state: dict) -> dict:**
    - Validate actor exists in state["actors"], return error if not
    - Validate arc_type (if provided) is in ARC_TYPES keys, return error if invalid
    - Validate arc_stage (if provided) is in ARC_STAGES keys, return error if invalid
    - Validate progress (if provided) is 0-100, return error if out of range
    - Support partial updates: only update provided fields, leave others unchanged (per D-25 discretion — set_actor_arc allows partial)
    - Initialize arc_progress with _DEFAULT_ARC_PROGRESS if not exists on actor
    - Return {"status": "success", "actor_name": ..., "arc_progress": {...}}
