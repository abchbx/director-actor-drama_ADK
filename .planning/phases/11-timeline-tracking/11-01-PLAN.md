```python
        flashback: Whether this is a flashback scene (D-05).

    Returns:
        dict with status, message, and updated timeline info.
    """
```
Logic:
- If day is None or period is None, call `parse_time_description(time_description)` to fill gaps
- If day is still None after parse, update only `current_time` string, return `{"status": "info", "message": "⚠️ 无法解析天数..."}`
- If period is provided but not in TIME_PERIODS, return error
- Update `state["timeline"]["current_time"] = time_description`
- Update `state["timeline"]["days_elapsed"] = day` (if day is not None)
- Update `state["timeline"]["current_period"] = period` (if period is not None)
- Append to `state["timeline"]["time_periods"]`:
  ```python
  {
      "label": time_description,
      "day": day,
      "period": period,
      "scene_range": [current_scene, current_scene],
      "flashback": flashback,
  }
  ```
- If len(time_periods) > MAX_TIME_PERIODS, call `_merge_oldest_time_periods(state)` to merge same-day entries
- Auto-run `detect_timeline_jump_logic(state)` and store result in `state["timeline"]["last_jump_check"]`
- Return `{"status": "success", "message": "⏰ 时间推进至...", "timeline": state["timeline"]}`

```python
def detect_timeline_jump_logic(state: dict) -> dict:
    """Detect timeline jumps by comparing adjacent time_periods day values.

    跳跃检测逻辑——对比 time_periods 中相邻条目的 day 差值（D-11/D-12/D-13/D-14）。
    同一天时段变化→normal，跨1-2天→minor，跨3+天→significant。
    不自动修正，返回提醒和建议（D-12）。

    Args:
        state: Drama state dict with timeline.

    Returns:
        dict with status, jumps list, max_gap.
    """
```
Implementation:
- Get `state["timeline"]["time_periods"]`
- If len < 2, return `{"status": "success", "jumps": [], "max_gap": 0}`
- Iterate pairs of consecutive time_periods
- Skip pairs where either entry has `"flashback": True` (D-05)
- Calculate `day_gap = abs(b["day"] - a["day"])` (handle None day with 0)
- Determine severity:
  - day_gap == 0 → "normal"
  - day_gap <= TIMELINE_JUMP_THRESHOLDS["significant"] - 1 (i.e. 1-2) → "minor"
  - day_gap >= TIMELINE_JUMP_THRESHOLDS["significant"] (i.e. 3+) → "significant"
- Build suggestion messages per severity level:
  - normal: "同一天时段变化"
  - minor: f"跨 {day_gap} 天无过渡，建议补充时间过渡说明"
  - significant: f"跨 {day_gap} 天无过渡，建议插入过渡场景或用旁白说明时间流逝"
- Build jump dicts per D-13:
  ```python
  {
      "from_scene": a["scene_range"][-1],
      "to_scene": b["scene_range"][0],
      "from_time": {"day": a["day"], "period": a["period"]},
      "to_time": {"day": b["day"], "period": b["period"]},
      "day_gap": day_gap,
      "severity": severity,
      "suggestion": suggestion,
  }
  ```
- Only include jumps with severity "minor" or "significant" (skip "normal" — no need to report same-day changes)
- Return `{"status": "success", "jumps": jumps, "max_gap": max_gap}`
