## Overview

Director-Actor-Drama 无限畅写版 — 基于 Google ADK 的多智能体即兴戏剧系统。核心承诺：**无限畅写，逻辑不断**。剧情可无限延续，每场与前文逻辑连贯、情感连贯、记忆连贯。

## Milestone Summary

**Key Accomplishments:**
1. 3 层记忆架构 + 异步压缩 — 支撑 50+ 场戏不溢出上下文
2. 无限叙事循环 + 混合推进模式 — AI 自主推进 + 用户随时干预无缝切换
3. 张力评分 + 冲突引擎 + Dynamic STORM — 自动检测平淡并注入转折
4. 一致性检查 + 矛盾修复 + 时间线追踪 — 保障"逻辑不断"核心承诺
5. 端到端集成 + 517 测试通过 — 全流程验证可交付

**Key Decisions:**
- 混合推进模式：用户需要自由度，纯 AI 推进会失控，纯用户驱动会中断沉浸感 ✓
- 动态 STORM 替代静态 STORM：静态大纲限制剧情长度，动态视角发现让剧情可无限生长 ✓
- 分层记忆而非全量记忆：上下文窗口有限，全量记忆 50 场后不可行 ✓
- 保留 A2A 进程隔离：认知边界隔离是核心价值，不可为了性能牺牲 ✓
- 保留文件系统持久化：JSON 文件持久化简单可靠 ✓

**Issues Resolved:**
- actor_speak() UnboundLocalError 修复
- conversation_log 全局状态迁移至 state
- 场景归档防止 state.json 无限膨胀
- 共享 AsyncClient 解决连接泄漏
- 崩溃恢复：被动检测 + 自动重启 + MAX_CRASH_COUNT=3

**Technical Debt Incurred:**
- `_current_drama_folder` 全局变量迁移未完成（D-07 TODO）
- E2E 测试需要真实 LLM API
